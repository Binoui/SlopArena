Menu Flow

How should the screens connect? Choose the overall navigation structure.

A

Linear Funnel

Main Menu → Char Select → Stage Select → Match

Both players pick on the same screen. Works for local co-op and solo training. Multiplayer just routes to the same funnel after host/join.

B

Lobby First

Main Menu → Lobby → (everyone picks char) → Stage vote → Match

Multiplayer-native: char select happens inside the lobby after connection, so both clients are synced before the match loads. More setup for training mode.

C

Mode-Split

Training: Main → Char → Stage → Match
PvP: Main → Host/Join → Lobby → Char → Stage → Match

Each mode has its own dedicated flow. More screens to build, but each path is clean and purpose-built.
